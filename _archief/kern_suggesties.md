# Kern-suggesties per Leerpad — Review v2

Dit zijn de kern-suggesties die de ruggengraat van de leercurve vormen.
Elke suggestie staat in het format zoals de app het gebruikt.
De `tekst` is wat Lisanne te zien krijgt (warme, begeleidende toon).

Let op:
- Stap-nummers zijn INTERN (Lisanne ziet ze niet, alleen thema-label)
- `inzicht` = korte uitleg waarom dit werkt (verschijnt als Lisanne op 'meer info' tikt)
- `reflectieVraag` = verschijnt nadat ze feedback geeft
- **KERN** = veilig, herhaalbaar, relationeel — altijd toonbaar
- **STRETCH** = alleen tonen na meerdere positieve ervaringen in dat leerpad
